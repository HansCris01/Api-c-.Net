Api crud con c# .NET y SqlServer


paso 1:

_ Crear la base de datos y la tabla:


---
CREATE TABLE Productos (
    Id INT PRIMARY KEY IDENTITY(1,1),
    Nombre NVARCHAR(255) NOT NULL,
    Precio DECIMAL(8, 2) NOT NULL,
    Stock INT NOT NULL
);
